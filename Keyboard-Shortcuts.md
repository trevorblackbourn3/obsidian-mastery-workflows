## Platform Key Notation

**Throughout this appendix:**

- **Ctrl** = Windows/Linux Control key
- **Cmd** = Mac Command key
- **Alt** = Windows/Linux Alt key
- **Opt** = Mac Option key
- **Shift** = Shift key (same on all platforms)

**Mac users:** Replace `Ctrl` with `Cmd` in most shortcuts below.

## Setting Custom Shortcuts

**To customize any shortcut:**

1. Open Settings (`Ctrl/Cmd + ,`)
2. Click "Hotkeys" in left sidebar
3. Search for command
4. Click in "Hotkey" field
5. Press desired key combination
6. Close settings

## Shortcut Conflicts

**If shortcut doesn't work:**

1. Check for conflicts: Settings → Hotkeys → Filter by shortcut
2. Unassign conflicting command
3. Reassign to your preferred action

**Common conflicts:**
- `Ctrl/Cmd + E` → Edit mode vs. plugin commands
- `Ctrl/Cmd + P` → Command palette vs. print (browser)
- `Ctrl/Cmd + D` → Select word vs. daily note (if set)

## Platform-Specific Differences

### Windows vs. Mac

**Most shortcuts translate directly:**
- `Ctrl` on Windows = `Cmd` on Mac
- `Alt` on Windows = `Opt` on Mac

**Exceptions:**
- Tab switching: `Ctrl + Tab` (Windows) vs. `Cmd + Opt + →/←` (Mac)
- Some system shortcuts override (Mac uses `Cmd + H` to hide, etc.)

---

### Linux

**Same as Windows shortcuts, with exceptions:**
- Desktop environment may override some shortcuts
- Check for conflicts with system/window manager shortcuts

## Troubleshooting Shortcuts

### Shortcut Not Working

**Check:**
1. Is plugin that provides command enabled?
2. Are you in correct mode (edit vs. preview)?
3. Is there a shortcut conflict?
4. Does your OS override this shortcut?

**Solution:**
- Settings → Hotkeys → Search for command
- Check if shortcut is assigned
- Look for conflicts
- Reassign if needed

---

### Forgot a Shortcut

**Use command palette:**
1. `Ctrl/Cmd + P`
2. Type command name
3. Execute command
4. Command palette shows shortcut next to command name (reminder)
